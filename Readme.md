# BMI Calculator

A simple CLI tool to calculate and interpret Body Mass Index (BMI) from user input, using Python.

## Features

- Validates user input for positive values only.
- Calculates BMI using the standard formula:  
The BMI is given by $\text{BMI} = \frac{\text{Weight (kg)}}{(\text{Height (m)})^2}$.


- Interprets BMI based on WHO classification:
  - Underweight: BMI < 18.5
  - Normal weight: 18.5 ≤ BMI < 25
  - Overweight: 25 ≤ BMI < 30
  - Obesity: BMI ≥ 30
- User-friendly and robust error handling.

## Requirements

- Python 3.x

No extra packages needed.
